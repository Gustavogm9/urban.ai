# Auditar Código e Merge Requests
**Agente Responsável:** Vitor Verificador (Auditor de Código)
**Descrição:** O Vitor inicia a análise dos tickets "Concluídos" no board da sprint e busca por branches associados, PRs ou commits que contenham o ticket ID. Caso o ticket afirme "Feito" mas nenhum código seja encontrado, a tarefa é imediatamente flagrada como "Pendente de Validação Técnica" e removida dos itens a serem atualizados nesta run.
